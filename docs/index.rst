TAXII 2 Python API Documentation
================================

*taxii2client* is a minimal implementation of a TAXII 2.0 Client in Python. It
currently supports the current services:

- Server Discovery
- Get API Root Information
- Get Status
- Get Collections
- Get a Collection
- Get Objects
- Add Objects
- Get an Object
- Get Object Manifests
